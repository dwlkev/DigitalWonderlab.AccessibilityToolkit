What do visual checks currently cover?

Visual checks currently focus on computed color contrast issues. They complement server-side HTML checks.

Why might screenshot previews be missing?

Some pages block capture due to browser security constraints (for example cross-origin assets). The report still includes selectors and contrast details.

How does licensing work?

Core functionality is Free Forever. Pro features are currently enabled as an Early Adopter trial.

Does this guarantee WCAG AA compliance?

This toolkit helps you work toward WCAG AA, but automated scans are not fully exhaustive and do not replace expert review. If you need deeper coverage, visit the Help & Services tab.

How can I report issues or request features?

Use the GitHub issues tracker: Accessibility Toolkit Issues
